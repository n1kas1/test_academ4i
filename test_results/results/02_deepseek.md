## Условие

Вычислите неопределённый интеграл $\displaystyle\int\left(2x^3 - \frac{5}{x} + e^{x}\right)dx$.

## Эталонный ответ

$\dfrac{x^4}{2} - 5\ln|x| + e^{x} + C$

## Ответ модели (raw)

```
\hd{Условие}
Вычислить неопределённый интеграл $\displaystyle\int\left(2x^3 - \frac{5}{x} + e^{x}\right)dx$.

\hd{Что найти}
Первообразную функции $f(x)=2x^3-\dfrac{5}{x}+e^{x}$, т.е. функцию $F(x)$ такую, что $F'(x)=f(x)$, записанную в виде $\displaystyle\int f(x)\,dx$.

\hd{Метод}
Используем свойство линейности интеграла и табличные интегралы: интеграл от суммы равен сумме интегралов, постоянный множитель можно выносить за знак интеграла.

\hd{Решение}
\textbf{Шаг 1.} Применяем свойство линейности, разбивая интеграл от суммы на сумму интегралов:
\[
\int\left(2x^3 - \frac{5}{x} + e^{x}\right)dx = \int 2x^3\,dx - \int \frac{5}{x}\,dx + \int e^{x}\,dx.
\]

\textbf{Шаг 2.} Выносим постоянные множители за знаки интегралов:
\[
= 2\int x^3\,dx - 5\int \frac{1}{x}\,dx + \int e^{x}\,dx.
\]

\textbf{Шаг 3.} Вычисляем каждый интеграл по таблице первообразных:
\begin{align*}
\int x^3\,dx &= \frac{x^{4}}{4} + C_1,\\
\int \frac{1}{x}\,dx &= \ln|x| + C_2,\\
\int e^{x}\,dx &= e^{x} + C_3,
\end{align*}
где $C_1, C_2, C_3$ — произвольные постоянные.

\textbf{Шаг 4.} Подставляем полученные выражения, объединяя все произвольные постоянные в одну $C$:
\[
2\cdot\frac{x^{4}}{4} - 5\ln|x| + e^{x} + C = \frac{x^{4}}{2} - 5\ln|x| + e^{x} + C.
\]

\hd{Ответ}
\ans{$\displaystyle \frac{x^{4}}{2} - 5\ln|x| + e^{x} + C$}
```
