## Условие

Найдите производную функции $y = \ln\bigl(\sin(x^2)\bigr)$.

## Эталонный ответ

$y' = \dfrac{2x\cos(x^2)}{\sin(x^2)} = 2x\,\operatorname{ctg}(x^2)$

## Ответ модели (raw)

```
\hd{Условие}

Найти производную функции $y = \ln\bigl(\sin(x^2)\bigr)$.

\hd{Что найти}

$y' = \dfrac{d}{dx}\ln\bigl(\sin(x^2)\bigr)$

\hd{Метод}

Применяем правило дифференцирования сложной функции (цепное правило) трёхкратно:
$$y = f(g(h(x))), \quad y' = f'(g(h(x)))\cdot g'(h(x))\cdot h'(x).$$

Здесь $f(u) = \ln u$, $g(v) = \sin v$, $h(x) = x^2$.

\hd{Решение}

\textbf{Шаг 1.} Запишем структуру сложной функции и производные каждого звена.

Внешняя функция: $f(u) = \ln u$, поэтому $f'(u) = \dfrac{1}{u}$.

Средняя функция: $g(v) = \sin v$, поэтому $g'(v) = \cos v$.

Внутренняя функция: $h(x) = x^2$, поэтому $h'(x) = 2x$.

\textbf{Шаг 2.} Применяем цепное правило.

$$y' = \frac{1}{\sin(x^2)} \cdot \cos(x^2) \cdot 2x$$

\textbf{Шаг 3.} Упрощаем, используя тождество $\dfrac{\cos v}{\sin v} = \ctg v$.

\begin{align*}
y' &= \frac{2x\cos(x^2)}{\sin(x^2)} = 2x\ctg(x^2).
\end{align*}

\hd{Ответ}
\ans{y' = 2x\,\ctg(x^2)}
```
