\hd{Условие}
Вычислить несобственный интеграл $\displaystyle\int_{-\infty}^{+\infty}\frac{dx}{(x^2+1)^2}$ с помощью теории вычетов.

\hd{Что найти}
Значение интеграла $\displaystyle I = \int_{-\infty}^{+\infty}\frac{dx}{(x^2+1)^2}$.

\hd{Метод}
Используем метод вычисления интегралов рациональных функций с помощью теории вычетов, рассматривая интеграл по вещественной оси как предел интеграла по замкнутому контуру в верхней полуплоскости.

\hd{Решение}

\textbf{Шаг 1.} Рассмотрим функцию комплексного переменного
$$f(z) = \frac{1}{(z^2+1)^2} = \frac{1}{(z-i)^2(z+i)^2}.$$
Особые точки функции: $z = i$ (полюс второго порядка) и $z = -i$ (полюс второго порядка).

\textbf{Шаг 2.} Выберем замкнутый контур $C$, состоящий из отрезка вещественной оси $[-R, R]$ и полуокружности $\Gamma_R$ радиуса $R$ в верхней полуплоскости. При $R > 1$ внутри контура находится только особая точка $z = i$.

\textbf{Шаг 3.} По основной теореме о вычетах:
$$\oint_C f(z)\,dz = 2\pi i \cdot \operatorname{res}_{z=i} f(z).$$

\textbf{Шаг 4.} Вычислим вычет функции $f(z)$ в точке $z = i$ (полюс второго порядка):
$$\operatorname{res}_{z=i} f(z) = \frac{1}{(2-1)!} \lim_{z\to i} \frac{d}{dz}\left[(z-i)^2 f(z)\right] = \lim_{z\to i} \frac{d}{dz}\left[\frac{1}{(z+i)^2}\right].$$
$$= \lim_{z\to i} \left(-\frac{2}{(z+i)^3}\right) = -\frac{2}{(2i)^3} = -\frac{2}{-8i} = \frac{1}{4i}.$$

\textbf{Шаг 5.} Представим интеграл по контуру:
$$\oint_C f(z)\,dz = \int_{-R}^{R} f(x)\,dx + \int_{\Gamma_R} f(z)\,dz.$$
При $R \to \infty$ интеграл по полуокружности стремится к нулю, так как
$$|f(z)| = \frac{1}{|z^2+1|^2} \leq \frac{1}{(R^2-1)^2} \quad \text{для } |z| = R,$$
и длина дуги $\pi R$, поэтому
$$\left|\int_{\Gamma_R} f(z)\,dz\right| \leq \frac{\pi R}{(R^2-1)^2} \to 0 \quad \text{при } R \to \infty.$$

\textbf{Шаг 6.} Переходя к пределу при $R \to \infty$, получаем:
$$\int_{-\infty}^{+\infty} f(x)\,dx = 2\pi i \cdot \operatorname{res}_{z=i} f(z) = 2\pi i \cdot \frac{1}{4i} = \frac{\pi}{2}.$$
