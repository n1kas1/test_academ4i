## Условие

Вычислите неопределённый интеграл $\displaystyle\int\left(2x^3 - \frac{5}{x} + e^{x}\right)dx$.

## Эталонный ответ

$\dfrac{x^4}{2} - 5\ln|x| + e^{x} + C$

## Ответ модели (raw)

\hd{Решение}

\textbf{Шаг 1.} Разбиваем интеграл на сумму трёх интегралов по линейности:
$$\int\!\left(2x^3 - \frac{5}{x} + e^{x}\right)dx

$$2\int x^3\,dx = 2\cdot\frac{x^{3+1}}{3+1} = 2\cdot\frac{x^4}{4} = \frac{x^4}{2}.$$

\textbf{Шаг 3.} Вычисляем второй интеграл по табличной формуле:
$$-5\int \frac{dx}{x} = -5\ln|x|.$$

\textbf{Шаг 4.} Вычисляем третий интеграл:
$$\int e^x\,dx = e^x.$$

\textbf{Шаг 5.} Объединяем результаты, добавляя произвольную постоянную $C$:
$$\int\!\left(2x^3 - \frac{5}{x} + e^{x}\right)dx = \frac{x^4}{2} - 5\ln|x| + e^x + C.$$
