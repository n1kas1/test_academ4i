## Условие

Решите рекуррентное соотношение $a_n = 3a_{n-1} - 2a_{n-2}$ с начальными условиями $a_0 = 2$, $a_1 = 3$. Найдите формулу общего члена.

## Эталонный ответ

$a_n = 1 + 2^{n}$

## Ответ модели (raw)

```
\hd{Условие}

Дано рекуррентное соотношение $a_n = 3a_{n-1} - 2a_{n-2}$ с начальными условиями $a_0 = 2$, $a_1 = 3$.

\hd{Что найти}

Формулу общего члена $a_n$ в замкнутом виде.

\hd{Метод}

Решение линейного однородного рекуррентного соотношения с постоянными коэффициентами через характеристическое уравнение. Общее решение — линейная комбинация частных решений вида $r^n$.

\hd{Решение}

\textbf{Шаг 1. Составим характеристическое уравнение.}

Перепишем соотношение в стандартном виде:
$$a_n - 3a_{n-1} + 2a_{n-2} = 0.$$
Подставляем $a_n = r^n$ и делим на $r^{n-2}$:
$$r^2 - 3r + 2 = 0.$$

\textbf{Шаг 2. Найдём корни характеристического уравнения.}

$$r^2 - 3r + 2 = (r - 1)(r - 2) = 0,$$
откуда $r_1 = 1$, $r_2 = 2$.

Корни различны, поэтому общее решение имеет вид:
$$a_n = C_1 \cdot 1^n + C_2 \cdot 2^n = C_1 + C_2 \cdot 2^n.$$

\textbf{Шаг 3. Применим начальные условия для нахождения констант.}

При $n = 0$:
$$a_0 = C_1 + C_2 = 2.$$

При $n = 1$:
$$a_1 = C_1 + 2C_2 = 3.$$

Получаем систему:
$$\begin{cases} C_1 + C_2 = 2, \\ C_1 + 2C_2 = 3. \end{cases}$$

\textbf{Шаг 4. Решим систему.}

Вычтем первое уравнение из второго:
$$(C_1 + 2C_2) - (C_1 + C_2) = 3 - 2 \implies C_2 = 1.$$

Подставим обратно в первое уравнение:
$$C_1 + 1 = 2 \implies C_1 = 1.$$

\textbf{Шаг 5. Запишем формулу общего члена.}

$$a_n = 1 \cdot 1^n + 1 \cdot 2^n = 1 + 2^n.$$

\textbf{Проверка:}
\begin{align*}
a_0 &= 1 + 2^0 = 1 + 1 = 2 \checkmark \\
a_1 &= 1 + 2^1 = 1 + 2 = 3 \checkmark \\
a_2 &= 3 \cdot 3 - 2 \cdot 2 = 9 - 4 = 5, \quad 1 + 2^2 = 5 \checkmark
\end{align*}

\hd{Ответ}
\ans{a_n = 1 + 2^n}
```
