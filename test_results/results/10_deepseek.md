## Условие

Исследуйте на сходимость ряд $\displaystyle\sum_{n=1}^{\infty}\sin\bigl(\pi\sqrt{n^2+1}\bigr)$.

## Эталонный ответ

Сходится условно. $\sqrt{n^2+1} = n + \frac{1}{2n} + O(n^{-3})$, поэтому $\sin(\pi\sqrt{n^2+1}) = (-1)^n \sin\!\left(\frac{\pi}{2n} + O(n^{-3})\right) \sim (-1)^n \frac{\pi}{2n}$ — знакочередующийся ряд, сходится по Лейбницу; абсолютно расходится (как $\sum 1/n$).

\hd{Решение}
Преобразуем выражение под знаком синуса. Сначала выделим целую часть:
\[
\pi\sqrt{n^2+1}= \pi n\sqrt{1+\frac{1}{n^2}}.
\]
Разложим корень по формуле Тейлора при $n \to \infty$:
\[
\sqrt{1+\frac{1}{n^2}} = 1 + \frac{1}{2n^2} + o\!\left(\frac{1}{n^2}\right).
\]
Тогда
\[
\pi\sqrt{n^2+1}= \pi n + \frac{\pi}{2n} + o\!\left(\frac{1}{n}\right).
\]
Обозначим $\alpha_n = \frac{\pi}{2n} + o\!\left(\frac{1}{n}\right)$. Тогда
\[
\sin\!\bigl(\pi\sqrt{n^2+1}\bigr) = \sin\!\bigl(\pi n + \alpha_n\bigr).
\]
Используем формулу приведения: $\sin(\pi n + \alpha_n) = (-1)^n \sin \alpha_n$ (поскольку $\sin(\pi n + x) = (-1)^n \sin x$). Поэтому
\[
a_n := \sin\!\bigl(\pi\sqrt{n^2+1}\bigr) = (-1)^n \sin \alpha_n.
\]
При больших $n$ величина $\alpha_n$ мала, поэтому $\sin \alpha_n \sim \alpha_n$. Более точно:
\[
\alpha_n = \frac{\pi}{2n} + \beta_n, \quad \text{где } \beta_n = o\!\left(\frac{1}{n}\right).
\]
Тогда
\[
\sin \alpha_n = \alpha_n - \frac{\alpha_n^3}{6} + o(\alpha_n^3) = \frac{\pi}{2n} + \beta_n - \frac{1}{6}\left(\frac{\pi}{2n} + \beta_n\right)^3 + o\!\left(\frac{1}{n^3}\right).
\]
Следовательно, при $n \to \infty$ справедливо асимптотическое равенство
\[
\sin \alpha_n = \frac{\pi}{2n} + o\!\left(\frac{1}{n}\right).
\]
Значит,
\[
a_n = (-1)^n \cdot \frac{\pi}{2n} + b_n, \quad \text{где } b_n = (-1)^n \cdot o\!\left(\frac{1}{n}\right).
\]
Ряд $\sum_{n=1}^{\infty} \frac{(-1)^n}{n}$ сходится по признаку Лейбница (как знакочередующийся гармонический ряд). Ряд $\sum_{n=1}^{\infty} b_n$ также сходится абсолютно, поскольку $b_n = O\!\left(\frac{1}{n^{1+\varepsilon}}\right)$ для некоторого $\varepsilon>0$ (например, можно считать $b_n$ эквивалентен $C \cdot n^{-3/2}$ с точностью до умножения на $(-1)^n$), что следует из уточнения разложения: $\beta_n = O(1/n^2)$, а кубический член $O(1/n^3)$. Поэтому $\sum b_n$ сходится абсолютно. Тогда исходный ряд $\sum a_n$ представляет собой сумму двух сходящихся рядов, значит, он сходится.

Проверим корректность: общий член имеет вид $(-1)^n \sin\!\bigl(\frac{\pi}{2n} + o(1/n)\bigr)$. Значит, ряд сходится.
